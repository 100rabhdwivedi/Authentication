# 🔐 Secure Authentication System

A modern authentication system demonstrating **production-level security practices** using the MERN stack.

## 🚀 Features

* Secure User Registration & Login
* Password Hashing with **Bcrypt**
* **JWT Access & Refresh Tokens** for authentication
* **Protected Routes** (Frontend & Backend)
* **Role-Based Authorization** (Admin / User)
* **Two-Factor Authentication (2FA)** with OTP
* **Rate Limiting** (IP & Email) to prevent brute-force attacks
* **CSRF Protection** using tokens
* **Input Sanitization** to prevent NoSQL Injection
* **Secure Cookie Management** (`httpOnly`, `secure`)
* Modern and responsive **React UI**

## 🛠 Tech Stack

* React
* Node.js
* Express.js
* MongoDB
* JWT
* Bcrypt

## ⚙️ Setup

```bash
git clone https://github.com/100rabhdwivedi/Authentication
cd Authentication
npm install
npm run dev
```
